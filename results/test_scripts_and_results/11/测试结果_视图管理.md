## 测试点3：列表视图列管理

| 项目 | 内容 |
|------|------|
| **Given** | 已登录，处于列表视图中 |
| **When** | 1. 点击列表右上角"编辑列表视图"省略号菜单<br>2. 在弹出的菜单中点击"选择列"<br>3. 取消勾选"标签"列将其隐藏 |
| **Then** | 列管理菜单正常打开，隐藏"标签"列后列表视图不再显示该列 |

| 项目 | 内容 |
|------|------|
| **操作步骤** | 1. 点击"编辑列表视图"按钮 → 弹出菜单（含选择列、重置列显示、重置列宽、适应屏幕宽度、重置列排序）<br>2. 点击"选择列" → 弹出列选择器，显示所有可选列及其勾选状态<br>3. 取消勾选"标签"列 → 列表视图中的"标签"列被隐藏 |
| **结果** | 列管理功能正常，列选择器菜单正确显示所有可用列，可成功隐藏/显示列 |
| **通过** | ✅ 通过 |
| **截图** | column_menu.png / column_adjusted.png |
